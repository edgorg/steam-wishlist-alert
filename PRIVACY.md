# Privacy Policy - Steam Wishlist Alerts

## Data Collection

Steam Wishlist Alerts does not collect, transmit, or store any personal data on external servers. All data remains locally in your browser.

## What Is Stored Locally

- Game IDs and names you choose to track
- Price targets you set
- Your selected region and theme preferences
- Price data fetched from Steam's public API

## External Connections

The extension communicates only with:
- store.steampowered.com (game prices and search)
- steamcommunity.com (profile resolution for wishlist import)
- cdn.akamai.steamstatic.com (game images)

No data is sent to any other server.

## Permissions

All permissions are used solely for core functionality. See the README for detailed justifications.
